
자유게시판(BOARD1TBL) 목록 닉네임 or 기업명 표시용 view 생성
CREATE OR REPLACE VIEW BOARD1MEMBERVIEW AS
SELECT T1.BNO, T1.MPNICKNAME, MCNAME  FROM
(SELECT * FROM BOARD1TBL LEFT OUTER JOIN MEMBERPERSONALTBL ON BOARD1TBL.MID = MEMBERPERSONALTBL.M_ID) T1 LEFT OUTER JOIN MEMBERCOMPANYTBL T2 ON T1.MID = T2.M_ID;

1. 클럽에 가입한 회원의 소속 클럽 번호, 아이디, 가입일, 등급코드(scode), 등급내용(scontent) 조회하는 view (CREGLISTVIEW) 생성 
CREATE OR REPLACE VIEW CREGLISTVIEW AS SELECT * FROM
(SELECT T1.CNO, T1.M_ID, T1.JCDATE, T1.S_CODE, T2.SCONTENT FROM (SELECT * FROM CLUB LEFT OUTER JOIN JOINCLUBTBL ON CLUB.CNO = JOINCLUBTBL.C_NO) T1 LEFT OUTER JOIN STEPTBL T2 ON T1.S_CODE = T2.SCODE); 

2. ↑에서 생성한 view의 내용과 각 회원의 닉네임 또는 기업명 조회하는 view 생성 (CREGLISTVIEW를 먼저 생성하고 이 view를 생성해야 됨)
CREATE OR REPLACE VIEW CREGLISTMEMVIEW AS SELECT * FROM
(SELECT T1.*, T2.MCNAME FROM (SELECT CRLV.*, MPT.MPNICKNAME FROM CREGLISTVIEW  CRLV LEFT OUTER JOIN MEMBERPERSONALTBL MPT ON CRLV.M_ID = MPT.M_ID) T1 LEFT OUTER JOIN MEMBERCOMPANYTBL T2 ON T1.M_ID = T2.M_ID);

 C.CNAME, C.CDESC, C.CAREA, C.CIMAGEURL FROM CLUB AS C, CATEGORYTBL AS G,
